## 💻 Como Executar

1. Clone o repositório
```bash
git clone https://github.com/nicolasbaggio/EstudosJava.git
```

2. Navegue até o diretório do projeto
```bash
cd EstudosJava
```

3. Os exercícios estão organizados por pacotes, seguindo a estrutura do curso DevDojo
